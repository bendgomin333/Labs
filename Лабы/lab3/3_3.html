Цепочка вызовов

Напишите функцию sum, которая будет работать так: sum(1)(2) == 3; // 1 + 2 sum(1)(2)(3) == 6; // 1 + 2 + 3 sum(5)(-1)(2) == 6 sum(6)(-1)(-2)(-3) == 0 sum(0)(1)(2)(3)(4)(5) == 1 Количество скобок может быть любым.

Листинг

var ladder = { step: 0, up: function() { this.step++; return this; }, down: function() { this.step--; return this; }, showStep: function() { alert( this.step ); } };
