Перепишите 'if' в '?'

Перепишите if с использованием оператора '?': if (a + b < 4) { result = 'Мало'; } else { result = 'Много'; }

Листинг

function ab_num() { var a = document.getElementById("a_num").value; var b = document.getElementById("b_num").value; var result = (parseInt(a) + parseInt(b)) < 4 ? 'Мало' : 'Много'; alert(result); }
