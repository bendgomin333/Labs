Получить знак числа

Используя конструкцию if..else, напишите код, который получает значение prompt, а затем выводит alert:
1, если значение больше нуля,
-1, если значение меньше нуля,
0, если значение равно нулю.

Листинг

function check_number() { var c_number=prompt('Введите число'); if (c_number > 0) { alert('1 (Число больше нуля)'); } else if (c_number < 0) { alert('-1 (Число меньше нуля)'); } else if (c_number = 0) { alert('0 (Число равно нулю)'); } else { alert('Это не число, попробуйте еще раз'); } }
