Переписать if'ы в switch

Перепишите код с использованием одной конструкции switch: var a = +prompt('a?', ''); if (a == 0) { alert( 0 ); } if (a == 1) { alert( 1 ); } if (a == 2 || a == 3) { alert( '2,3' ); }

Листинг

function Switch() { var a = +prompt('a?',''); switch (a) { case 0: alert(0); break; case 1: alert(1); break; case 2: case 3: alert('2,3'); break; } }

Результат

Введите значение (0,1,2,3)
