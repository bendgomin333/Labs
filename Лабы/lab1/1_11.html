Проверка if вне диапазона

Напишите условие if для проверки того факта, что age НЕ находится между 14 и 90 включительно.

Сделайте два варианта условия: первый с использованием оператора НЕ !, второй – без этого оператора.

Листинг

Вариант 1:

function AgeCheck1() { var age = parseInt(document.getElementById('Age1').value); if (age < 14 || age > 90) { alert('Да'); } else { alert('Нет'); } }

Вариант 2:

function AgeCheck2() { var age = parseInt(document.getElementById('Age1').value); if (!(age >= 14 && age <= 90)) { alert('Да'); } else { alert('Нет'); } }
